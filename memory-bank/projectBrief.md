# Mellow WEB

# Overview
Personal web music player written in React.
It is streaming application that connects to a backend API serving songs and metadata.
It also contains 'Notepad' module for taking notes (separate from music player).

## Features
- **Dual Playback Modes**:
	- Audio Player (default): Streams audio via API using YouTube videoId
	- YT Player (legacy): Direct YouTube player integration
- **Song Management**: Browse and play songs served from API
- **Tag System**: Organize and filter music by tags

## Tech Stack
- JavaScript (ES6+) with ongoing migration to TypeScript
- React
- React Redux
- React Router
- Ant Design (UI Library)

**Build Tools:**
- Webpack
- Yarn (package management)
